# String.prototype.toLowerCase

# 1. 语法

WARNING

str.toLowerCase()

返回值：

小写的字符串。

# 2. 描述

描述即返回值。

个人在日常开发中常用场景：

  • 用于小写字符串。

# 3. 示例

  • using toLowerCase

    'ALPHABET'.toLowerCase(); // 'alphabet'
